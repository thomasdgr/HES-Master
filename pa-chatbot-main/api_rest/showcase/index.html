Created by Thomas Dagier (03-2024 - 06.2024).

 

This microservice expose a chatbot API through a REST API. Made for the HES-SO MSE the project aims to answer questions based on several adminstrative documents brought by the HEIA-FR in order to help students and staff to find the information they need.

 

More details

 

The microservice expose a RAG system using LLM models. For further information, please refer to the documentation or contact thomas.dagierjo@hes-so.ch
